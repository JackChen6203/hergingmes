# Hermes Agent Hugging Face Space 一鍵部署工具

這是給一般使用者（小學生也能用）的 Windows 桌面工具。

## 最簡單用法

解壓縮後，點兩下：

```bat
00_先點我_一鍵啟動.bat
```

它會自動：

- 檢查 Python（沒有就自動安裝）
- 建立乾淨 `.venv`
- 安裝桌面工具需要的套件
- 打開 GUI

## GUI 裡面的做法

1. 貼上 Hugging Face Write Token
2. 確認 Space 名稱，例如 `davisc1/hermes-agent-space`
3. 按 `🚀 一鍵開始（小學生模式）`
4. 看到 OpenAI / Codex 授權時，照畫面登入
5. 完成後打開 Hugging Face Space URL

## 檔案結構

```
00_先點我_一鍵啟動.bat      ← 入口，點這個就好
01_啟動桌面工具.ps1         ← 自動環境準備（不用手動執行）
02_只安裝不啟動.bat         ← 只安裝環境，不開 GUI
03_已安裝後直接執行.bat     ← 已安裝好的人用這個
04_打包成執行檔.bat/.ps1    ← 打包成 .exe（進階）
05_使用說明.md              ← 本檔案
06_沒有Python請看這裡.md    ← 給完全沒有 Python 的人看

app/                        ← 程式核心（不用動）
├── main.py                 ← 程式入口
├── config.py               ← 設定常數
├── core/                   ← 業務邏輯
│   ├── environment.py      ← 環境檢查
│   ├── hf_ops.py           ← HF 操作
│   ├── hermes_ops.py       ← Hermes 操作
│   ├── space_builder.py    ← Space 檔案產生
│   └── deployer.py         ← 全流程編排
└── ui/                     ← 介面
    ├── app_window.py       ← 主視窗
    ├── theme.py            ← 深色主題
    └── widgets.py          ← 自訂元件
```

## 安全提醒

不要把以下內容貼給別人，也不要 commit：

- HF Token
- `.hermes/auth.json`
- `.codex/auth.json`
- `HERMES_AUTH_JSON_B64`
