# Hergingmes Hermes HF One-Click Deployer

This repository contains a Windows one-click launcher and HuggingFace Space runtime for deploying a Hermes-style agent with OpenAI Codex authentication support.

## Included

- Windows launcher: `先點我_一鍵啟動.bat`
- Desktop helper app: `_internal/app/`
- HuggingFace Space runtime: `hermes-agent-space/`
- Local Chinese setup notes: `_說明/`

## Security

Runtime secrets are intentionally not committed. Configure HuggingFace, Telegram, Discord, Cloudflare, Codex/OpenAI, and other service tokens through the launcher or your deployment environment.
